# VLTG Apparel

A modern streetwear e-commerce website designed with a bold, futuristic aesthetic. This project showcases a clean shopping experience, custom apparel features, and a responsive user interface suitable for modern fashion brands.

## ✨ Features

- Modern streetwear landing page
- Responsive design
- Custom shirt builder
- Product showcase
- Shopping cart interface
- Smooth animations
- Clean and minimal UI
- Mobile-friendly layout

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose

This project was created as a portfolio piece to demonstrate modern web design and front-end development for clothing brands and e-commerce businesses.
